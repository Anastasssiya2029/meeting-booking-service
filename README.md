# Meeting Booking Service

Интерактивный прототип сервиса для записи на онлайн-встречи.

## Дизайн
- Минимализм Apple с 3D дизайном
- Космическая тема (градиент от синего через фиолетовый к красному)
- Анимированный фон с летающими часами/₽/%
- Полная мобильная адаптация

## Функционал
- Система аутентификации с мультитенантностью
- 4 роли: Архитектор, Админ, Менеджер, Помощник
- Управление окошками менеджеров
- Подбор времени через фильтры (будни/выходные, утро/день/вечер)
- Календарь встреч со статусами
- Аналитика конверсий и продаж
- Управление школами и пользователями

## Роли и доступ
- **Архитектор** — видит все школы, клиентов и встречи всех менеджеров
- **Админ** — видит клиентов и встречи всех менеджеров своей школы
- **Помощник** — записывает клиентов, видит данные всех менеджеров
- **Менеджер** — видит только свои данные

## Структура Meeting
Содержит только `startTime` без `endTime` — продолжительность определяется менеджером.

## Технологии
- React + TypeScript
- Tailwind CSS v4
- Recharts (для аналитики)
- Motion (для анимаций)
- Sonner (для уведомлений)

## Бэкенд
Спецификация бэкенда находится в файле `BACKEND_SPECIFICATION.md` — включает SQL-схемы 8 таблиц, 50+ API endpoints, систему ролей, JWT-аутентификацию, бизнес-логику.

Пример бэкенда на Node.js/Express/PostgreSQL — в папке `backend-example/`.

## Запуск
```bash
npm install
npm start
```

## Настройка бэкенда
Установите переменную окружения `REACT_APP_API_URL` для подключения к реальному бэкенду.
Без неё приложение работает в демо-режиме с моковыми данными.
